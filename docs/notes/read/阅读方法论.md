## 两大原则

### 方向比速度更重要（向上或向下）

- 先题后文，只看题干，不看选项
- 定位词

  - 大写、小写、地名、时间、年代、数字
  - 名词、形容词
  - 题干中有位移性的词

- 感情色彩强烈的形容词、副词表示作者的态度方向（名词动词也可以）
- 阅读的本质：逻辑（句间、段间、题目间）

  - 相同

    - 没有转折就是相同

      - instead
      - also

    - 相同往往被忽视、但更重要

  - 相反

    - 有转折

      - but、however...
      - 时间相反，一切相反（重点是现在）
      - if虚拟语气
      - should do、must do、claim to
      - instead of

- 识别同义表达降低阅读难度

### 细节服从主旨

- 和中心思想最相关的，是最佳答案

  - 答案是比较出来的

## 一堆技巧

### 答案是找出来的

- 回到原文逐字比对

### 答案是比较出来的

- 两个答案通过比较选最佳

  - 是否接近主旨
  - 是否多次提到
  - 修饰词的程度

### 因果逻辑关系

- because、since、as、for、lest、by
- derive from、originate from、stem from、result from、owing to、due to、attribute to、contribute to
- 定语从句

### 标点符号

- 逗号【，】

  - 两个逗号之间，或一个逗号之后作为补充说明的成分

- 冒号【：】

  - 从抽象到具体，后面解释前面

- 分号【；】

  - 并列关系

    - =

- 引号【" "】

  - 引用
  - 反语、语义反转

    - "所谓的"

- 破折号【——】

  - 破折号里的插入语不是主要信息，但一定要看

- 问号【？】

  - 出现在句首，提示中心句

- 括号内容也要看

### 语言的二义性（有歧义）

- 不能通过语法解决，只能通过上下文推测

### 行文“套路”

- “少数派”原则

  - 作者的态度一般是少数人的看法

- 欲扬先抑，欲抑先扬

### 不能跳读

### 线性思维，说一是一，不要过度推理

## 六大题型

### 例证题

- 标志

  - example、case、illustate、demostate、to show、to___
  - 隐藏型：题干没体现，回到原文有举例

- 原则

  - 例子本身不重要，重要的是例子所支持的观点
  - 观点一般在例子前（后面也有可能，有时不在同一段）

- 干扰选项

  - 例子本身，就事论事

- 观点句出现的标志

  - 转折
  - 情态动词
  - 感情色彩强烈的形容词、副词

- 注意

  - 例子要看，帮助排除干扰选项
  - 注意例子从哪里开始，到哪里结束
  - such、this、those、these...+n.指代前文内容
    出现在例子后，说明这句话还是例子本身

### 中心思想题

- 标志

  - mainly about/discuss,the best title

- 方法

  - 串线法

    - 串联各段首末句
    - 注意转折

  - 中心句

    - 文章开头有提问，对它的回答
    - 一个独句段
    - 对“？”要敏感

  - 中心词

    - 反复出现的是重点，重点是考点
    - 注意同义替换

- 干扰选项排除方法

  - 验证答案

    - 范围太小，片面
    - 范围太大，回头作文

### 作者态度题

- 标志

  - attitude，deem，consider，tone，选项词

- 选项词汇总结

  - 正面

    - positive、optimistic、approval、supportive

  - 负面

    - negative、pessimistic、disapproval、critical、doubtful
      suspicious、skeptical、questionable、objectionable怀疑即否定

  - 客观中立

    - neutral、object、impartial、disinterested

  - 小墓碑

    - uninterested、unconcerned、carefree、indifferent、detached
    - confused、puzzled
    - biased、prejudice、contemptuous、subjective

- 当作者态度没有明确提出时

  - 文章框架串线
  - 感情色彩强烈的形容词、副词
  - 转折
  - 情态动词后

### 猜词题

- 标志

  - “......”(line, paragraph) most probably means...

- 方法

  - 通过定位好的句子，根据逻辑关系（句内或句间）找同义词或反义词

- 干扰选项特征

  - 字面意思理解；简单词的常用意思

### 推理判断题

- 标志

  - infer、learn from、imply、suggest、conclude

- 方法

  - 同义改写

    - 细节
    - 主旨
    - 逻辑

  - 逻辑取反

    - 喜欢在转折处出正确答案

### 细节题

- 干扰选项特征

  - 正反混淆

    - 正反义词
    - 主被动偷换
    - 因果倒置

  - 偷换概念
  - 答非所问
  - 不同内容的嫁接
  - 非最佳答案
  - 绝对化用词：only、must、exclusively、never、all、no、最高级（一般不选，除非文中明确提到）
  - 硬币的两面性，文中只强调一个方面，但选项强调了另一个方面
  - 过度推理
  - 问原因的干扰选项

    - 一般问主要原因，干扰为次要原因

- 正确选项特征

  - 同义改写
  - 与中心密切相关

    - 细节服从主旨

  - 语气缓和

    - some、may、partly、might

  - 抽象概括性
  - 两个相反的选项往往有正确选项（除了态度题）
  - 正确选项之间有密切联系，暗示文章中心，可以帮助检验答案、排除干扰

- 定位好的句子是解题的关键依据
